SONA Internship Recap

Generative architecture using BIM software

From a real-life dice roll to an immersive VR experience

The goal was to implement generative and random architectural generation using the at the time pretty rigid Dynamo software, the computational design addon of Revit.
The project, made during my internship in SONA, a Nantes based architecture studio. It was made with Felix Babey.
